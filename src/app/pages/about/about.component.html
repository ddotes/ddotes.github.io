About Us

Dotes is a diary/notes website with the goal of facilitating the creation, organization and writing of journals/notes by gathering all said features in an easy to use environment. This project is done by Ronald Kirk and Erfan Taghiabadi who were just aiming to create a project out of pure boredom. This page is here because we did not know what else to put. Also, this is the only page that is responsive for Galaxy Fold users.
